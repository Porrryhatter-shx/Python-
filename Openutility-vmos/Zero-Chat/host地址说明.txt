1. 本地测试（ 127.0.0.1  或  localhost ）如果你只是在本地计算机上测试聊天软件，可以使用以下地址'127.0.0.1'。
这是IPv4地址，表示本机的环回地址（loopback address），即指向本机。
localhost ：这是环回地址的域名形式，通常会解析为 127.0.0.1 。
在本地测试时，服务器和客户端都运行在同一台计算机上，因此使用  127.0.0.1  或  localhost  是最方便的。
2. 同一局域网内的其他设备如果你希望在局域网（LAN）内的其他设备上运行客户端并连接到服务器，需要使用服务器的实际局域网IP地址可通过以下方法获取：

Windows：
    1. 打开命令提示符(CMD)
    2. 输入'ipconfig'并回车
    3. 查找“IPv4 地址”，例如  192.168.1.100
macOS/Linux：
    1. 打开终端
    2. 输入  ifconfig  或  ip addr(取决于系统)
    3. 查找  inet  地址，例如  192.168.1.100
    
将服务器代码中的  host  地址设置为这个局域网IP地址，客户端就可以通过这个地址连接到服务器了。
